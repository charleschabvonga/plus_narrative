Additional helpers and methods frequently used throughout PlusNarrative WordPress projects.

== Description ==
The PlusNarrative plugin was developed to expand the functionality of existing and new WordPress projects.

The idea is to abstract frequently used logic and helpers into this plugin.

= Links =
* [Documentation](https://bitbuck.org/plusnarrativeteam/pn-wp-helpers-plugin/src/master/readme.txt)


== Installation ==

From your WordPress dashboard

1. **Visit** Plugins > Add New
2. **Upload** Plugins > Upload Plugin
3. **Activate** PlusNarrative WP Helpers from your plugins page
4. **Read** the documentation to [get started](https://bitbuck.org/plusnarrativeteam/pn-wp-helpers-plugin/src/master/readme.text)
